# BestOilApp  
**C# WinForms-кассир для автозаправки «BestOil» с мини-кафе**

![Авто-заправка](png/BestOilApp.png)

## Что делает
- Продаёт бензин двумя способами: «X литров» или «на Y рублей».  
- Подсчитывает чек по кафе: хот-доги, гамбургеры, картофель-фри, кола.  
- Показывает итог, запоминает дневную выручку и предлагает очистить форму для следующего клиента.  
- По закрытию выводит общий доход за день.  

## Установка и запуск
1. Склонируйте репозиторий  
   ```bash
   git clone https://github.com/yourname/BestOilApp.git
   cd BestOilApp
   ```
2. Откройте `BestOilApp.sln` в Visual Studio 2022+  
3. Соберите решение (`Ctrl+Shift+B`). EXE появится в `bin\Debug\BestOilApp.exe`  
4. Запустите `BestOilApp.exe` — программа не требует установки

## Экраны и элементы
| Группа | Назначение |
|--------|------------|
| **Автозаправка** | Выбор топлива, переключатель «Количество/Сумма», блок «К оплате» |
| **Мини-кафе** | CheckBox-ы товаров, поля количества, блок «К оплате» |
| **ВСЕГО к оплате** | Кнопка «Рассчитать» и финальная сумма |

## Особенности
- После расчёта таймер предложит очистить форму через 10 с.  
- Только цифры и «,» допускаются в полях ввода.  
- Все картинки (иконки топлива, кафе, смайлик) хранятся как Embedded Resources в `Properties.Resources`.  
- Дневная выручка обнуляется при каждом новом запуске программы.

## Пример использования
1. Выберите «А-95» → автоматически появится цена 64,22 руб.  
2. Переключитесь на «Сумма (руб.)», введите 1000 → в поле «К выдаче» появится 15,57 л.  
3. В кафе поставьте галочку «Хот-дог», введите 2 шт.  
4. Нажмите «Рассчитать» → итог: 1000 + 340 = 1340 руб.  
5. Через 10 с система спросит: «Очистить форму?». Нажмите «Да» — поля сбросятся, выручка сохранится.

## Сборка релиза
```bash
msbuild BestOilApp.sln /p:Configuration=Release
```
Готовый файл: `bin\Release\BestOilApp.exe` (один EXE, ресурсы встроены).
